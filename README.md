## Getting Started

```bash
git clone https://github.com/JoseAUM/safehome.git

cd safehome

npm i
```

- [Front-End Development](docs/FRONT-END.md)

**To run the app**

```bash
npm run dev
```

Once the app is running, can be accessed on [http://localhost:8080/](http://localhost:8080/)

**Testing**

```bash
npm run test
```
